# Container (container)
[docs](https://togglebyte.github.io/anathema-guide/templates/elements/container.html)

A container element.

The container accepts at most one child.
## Example
```
border [width: 16, height: 5]
    container [width: 1]
        text "ab"
```
```angular2html
┌──────────────┐
│a             │
│b             │
│              │
└──────────────┘
```
