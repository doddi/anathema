# Column (column)
[docs](https://togglebyte.github.io/anathema-guide/templates/elements/column.html)

Vertical layout of elements where all the elements are centred horizontally.

Column accepts multiple children.

## Example
### column
```
text "a"
border
  text "b"
  text "c"
```
```
a
┌─┐
│b│
└─┘
c
```
